## Project Introduction

An interactive mobile application developed with technologies such as increased reality in order to enable children to have a quality and productive journey during air travel. It draws flight routes between the cities chosen by users on the map of TÃ¼rkiye; It provides children with a fun and educational flight experience by offering word puzzle games that contain parameters such as enriched reality in each city and parameters such as scoring time.
